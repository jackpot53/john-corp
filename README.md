안녕하세요! 바이브코딩으로 만드는 포트폴리오입니다.

Next.js + Tailwind CSS로 만든 단일 페이지 개인 포트폴리오 사이트입니다.

## 시작하기

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 결과를 확인할 수 있습니다.

## 콘텐츠 수정하기

실제 프로필/소개/프로젝트/경력 정보는 `src/content/` 폴더의 파일들을 수정하면 반영됩니다.

- `src/content/profile.ts` — 이름, 직함, 소셜 링크
- `src/content/about.ts` — 소개 문단, 강점
- `src/content/projects.ts` — 프로젝트 목록
- `src/content/experience.ts` — 경력 목록
